# Find Prime Number

<h2>Objectives</h2>
<ul>
  <li>Let user enter a upper bound</li>
  <li>Then show all the prime numbers below the upper bound.</li>
</ul>

<h2>Learning Outcomes</h2>
<ul>
  <li>For-if-else</li>
  <li>Write definition</li>
  <li>Prompt user input</li>
  <li>Input convertion</li>
</ul>
